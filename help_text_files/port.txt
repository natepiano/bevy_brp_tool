Port to connect to [default: 15702]

USAGE:
  {{BIN_NAME}} --port <PORT> <command>
  {{BIN_NAME}} -p <PORT> <command>

DESCRIPTION:
  Specifies which port to use for BRP communication.
  Must match the port configured in your Bevy app.

EXAMPLES:
  {{BIN_NAME}} --port 15703 list
  {{BIN_NAME}} -p 8080 ready
  {{BIN_NAME}} --detached --port 15703
  {{BIN_NAME}} --port 15703 --info

DEFAULT BEHAVIOR:
  - Uses port 15702 if not specified
  - This matches Bevy's RemoteHttpPlugin::default() port exactly
  - Ensures compatibility with apps using only basic BRP setup
  - Can be overridden in both app and CLI

COMMON SCENARIOS:
  - Multiple apps: Use different ports for each
  - Firewall issues: Try higher port numbers
  - Docker/containers: May need port mapping

NOTE:
 - use {{BIN_NAME}} --brp to learn more about port configuration with your bevy app
